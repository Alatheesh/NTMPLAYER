# NTMPLAYER

A premium cinematic streaming web experience built with modern frontend design, immersive UI effects, and custom media controls.

## ✨ Features

- 🎬 Premium cinematic streaming interface
- 📺 Custom video player controls
- ⚡ Smooth modern UI animations
- 🔊 Advanced playback controls
- 📱 Responsive design for desktop and mobile
- 🌌 Ambient visual effects and overlays
- 📺 Picture-in-Picture support
- ⏩ Playback speed controls
- 🔗 Stream sharing support
- 🎞 Episode container system

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Video.js
